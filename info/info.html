OVERVIEW

Icons are great addition to any website, app, or service. Icons make it easier for users to navigate pages, preceive content, and identify where they are. Icons are also great way to highlight important content for user. In my learned opinion, icons can also make work more productive. Less time you need for looking at the right link, page, or form, more time you have for other important work. There is no denying that icons are part of productive interaction on internet.

EZ-SVG is a JavaScript- and Graphics -project with main purpose of providing easy access to common svg-icons for anyone working on a website-, app-, or service project. EZ-SVG contains a library of svg-icons that are stored inside JavaScript -file. From there, icons can be called on a document with CSS -class.

SETUP

EZ-SVG icon gallery is easy to setup with only 3 steps.

  • Download latest release of EZ-SVG.zip from GitHub.
  • Extract ez-svg.js file from ZIP to your project folder. Only the one file is needed.
  • Link the file to your project HTML document, at the end of <body> -section (inside).
    <script src="location/of/ez-svg.js"></script>
  • You can now use the icons.

HOW TO USE ICONS

  • Once you have uploaded and linked ez-svg.js file on your project, you can call EZ-icons on your HTML document with <i> -tag containing CSS-class. Each icon has an unique class-name. <i class="ez-brush"></i> = .

  • Second way is to link SVG-image(s) in external CSS -file. Package includes independend SVG -images. Create CLASS or ID for wanted icon and add external SVG -image as content. When this is done, you can use icon as follows <i class="comment-icon"></i>.

    Icon size will be dynamic and changes with parent element, but icon color will be static.

  • You can also use icons as image-elements. As SVG-images are string of code insted of an image, normal sharpness rules don't apply to them. SVG images scale up infinitely with orginal sharpness. Logo at the header of this page is an SVG-image.

    When using SVG as image-element, it must be manually sized with CSS and color can be modified inside the SVG-file. You can open it with text-editor.

MANUAL ICON SIZE

All icons can have manual size-class added to them. There are 5 different sizes available: S, M, L, XL, XXL. Since all icons are naturally M-size (1em), manual size-class works as multiplier to it. Manual size-class is easy to apply. E.g.

<i class="ez-brush ez-xl"></i> =
